🏆 勝ち筋（競争優位性）
1. ネットワーク効果
出品者増 → 購入者増 → 成果データ蓄積 → 信頼性向上の好循環。プラットフォームが成長するほど価値が指数的に増大。
2. データモート
成果トラッキングデータが蓄積されるほど、エージェント品質の比較精度が向上。後発は同量のデータを得るのに時間がかかる。
3. スイッチングコスト
成果履歴・分配設定がプラットフォームに紐づくため移行困難。レピュテーションと実績の蓄積が参入障壁に。
4. 成果連動モデル
購入者リスクゼロ（成果が出なければ課金なし）→ 導入ハードル最低。従来のSaaS/買い切りモデルとの圧倒的差別化。
5. コード保護
プラットフォーム内実行でIP保護 → 出品者が安心して出品。GPT Storeのようにコードが丸見えにならない。

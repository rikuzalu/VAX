事業定義

VAX — Verified Agent Exchange

AI労働の成果を分配するインフラ

エージェントを売るのではなく、エージェントの成果を流通させるプラットフォーム。

基本設計ルール

  • コード非公開 — プラットフォーム内実行のみ（コピー不可）
  • 買い切り禁止 — レベニューシェアのみ（継続収益モデル）
  • 成果定義必須 — KPI・計測方法・責任範囲を事前に定義

収益フロー

エージェント実行 → 成果発生 → プラットフォーム計測 → 収益分配

  • データ分析を自動化したい企業

エージェントカテゴリ

カテゴリ 成果指標（KPI）
Sales Agents リード数・商談数・売上
Content Agents 生成数・流入・CV
Data Agents 取得件数・工数削減

差別化ポイント

  • 成果連動型課金 — 買い切りなし、成果に応じた支払い
  • コード非公開 — コピー不可、知的財産を保護
  • 透明な成果トラッキング — リアルタイムで成果を可視化
  • 自動収益分配 — Stripe Connect連携で自動精算

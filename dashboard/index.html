🎯 Planner — 計画立案

PRDを解析し、タスクグラフを構築。依存関係を自動検出し、最適な実行順序を決定する。

能力: タスク分解、依存関係解析、優先度付け、スプリント計画
モデル: Claude Opus 4
ステータス: 全タスク計画完了

⚡ Generator — コード生成

Plannerの計画に基づき、実際のコードを生成・実装する。テスト駆動開発を採用。

能力: コード生成、テスト作成、リファクタリング、ドキュメント
モデル: Claude Sonnet 4
ステータス: 次スプリント待ち

🔍 Evaluator — 品質評価

生成されたコードを多角的に評価。受入基準の充足、セキュリティ、パフォーマンスを検証。

能力: コードレビュー、テスト実行、セキュリティ監査、性能分析
モデル: Claude Opus 4
ステータス: Generatorの出力待ち
エージェント運用ルール
  • 各エージェントは独立したコンテキストで動作し、成果物のみを共有する
  • Plannerが全タスクのDAGを構築し、依存関係の解決順序を決定
  • Generatorはテストファースト：まずテストを書き、次に実装コードを生成
  • Evaluatorは受入基準チェックリストに基づき合否判定（80%以上で合格）
  • 不合格の場合、フィードバックループでGeneratorに差し戻し（最大3回）
  • 全エージェントのログは監査証跡として保存
  • 人間のレビューはEvaluator合格後に実施（最終承認ゲート）

VERIFIED AGENT EXCHANGE — 投資家向け資料

エグゼクティブサマリー

Verified Agent Exchange (VAX) は、AIエージェントの品質保証付きマーケットプレイスである。 企業がAIエージェントを安心して導入できるよう、第三者評価・成果トラッキング・収益分配を一体化したプラットフォームを構築する。

AIエージェント市場は $5.4B（2024年）→ $47.1B（2030年）、CAGR 54% で成長中。 しかし品質基準の欠如が普及のボトルネックとなっている。VAXはこの課題を解決し、「信頼のインフラ」としてデファクトスタンダードを目指す。
